# Server Performance Stats

Bash script to analyse Linux server performance stats.

## Run the script
chmod +x server-stats.sh
./server-stats.sh

## Stats included
- CPU usage
- Memory usage (Free vs Used with %)
- Disk usage (Free vs Used with %)
- Top 5 processes by CPU
- Top 5 processes by memory
- OS version, uptime, load average, logged-in users, failed logins
